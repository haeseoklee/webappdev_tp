Lab 8: Pimp My Text

Lab 8: Pimp My Text - create the appropriate Javascript for Pimp My Text web page

The Pimp My Text web page allows user to type text into a box and pimp out the text with funny styling by clicking on UI controls.

The Pimp My Text page has a linked Javascript (pimpmytext.js) that will manipulate the text.

Lab 8 exercises

Download the pimpmytext.html and complete the following exercises.

  1. Create UI Elements
  2. JavaScript alert
  3. Hello World Button
  4. Bigger Pimpin' Button
  5. Bling Checkbox
  6. Snoopify
  7. Font Timer
  8. Unobtrusive JavaScript
  9. More Gangsta (extra point exercises)

Ex 2: JavaScript alert

Let's try a bit of JavaScript testing code that pops up an alert box. This is just to test that your browser is running your JavaScript file correctly, before we move on to tougher exercises.

  • Create a new pimpmytext.js file.
  • Put the following line of code into the file: 
    alert("Hello, world!");
  • Link your HTML page to the pimpmytext.js.
  • Refresh your page in the browser. Do you see the alert message?
    • If so, move on.
    • If not, double-check your script tag syntax or ask a TA for help.

Ex 3: Hello World Button

  • Modify your JS file to wrap the alert into a function.

  • Refresh your page in the browser. Click the button. Do you see the alert?

Ex 4: Bigger Pimpin'! Button

(See example screenshot on next slide.)

Modify your JS code so that when the user clicks Bigger Pimpin'! button, the text in the text area will get larger.

  • Make sure your text area has an id attribute so your JS code can communicate with it.
  • Modify your JS function so that it now changes the text area's font size to 24pt.
  • Hint: The syntax for accessing an element by ID and changing a style is: 
    $("elementID").style.propertyName = "value";
    • Use proper units, such as 24pt, and proper capitalization, such as backgroundColor .

Ex 7: Font Timer

  • Modify the Bigger Pimpin'! button. On button click, rather than setting the font size to 24pt, it should make the font in the text area 2pt larger than its current size.
    • When the first time the button is clicked, the text area's .style.fontSize property won't have a value. So, in this case, just use a default of 12pt.
    • For future clicks, read the font size such as 12pt and change it to a larger one like 14pt.
    • Hint: use the parseInt function.
  • Once that works, make it use a timer.
    • A single click should continually increase the font size by +2pt every 500ms.
    • Use the setInterval function. The timer should call the function you just wrote.

Ex 9: More Gangsta

Try adding extra features if you have time :

  • When the Bling button is checked, also set the page to have a background image: 
    https://selab.hanyang.ac.kr/courses/cse326/2019/labs/images/8/hundred.jpg
  • Add an Igpay Atinlay button that converts the text to Pig Latin. The rules:
    1. Words beginning in a consonant (or consonant cluster) have that consonant (or consonant cluster) moved to the end and -ay tacked on following.
    2. Words beginning in a vowel simply have -ay tacked on the end.
    • e.g, pig --> igpay, banana --> ananabay, trash --> ashtray
  • Add a Malkovitch button that causes words of >= 5 characters in length to be replaced with the word Malkovich.
