Ex 1: Create DB and tables

Create a database called college with the following tables.

  • Student table ( table name: student) has the following columns:
    • has student_id column that has interger type
    • has name column that is 10 bytes long
    • has year column that has smallest integer type
    • has dept_no column that has interger type
    • has major column that is 20 bytes long
    • has student_id as primary key
    • does not allow NULL in any column except major
    • default value for year is 1

Ex 1: Create DB and tables

Create the following table in the college DB.

  • Department table ( table name: department) has the following columns:
    • has dept_no column that has interger type
    • has dept_name column that is 20 bytes long
    • has office column that is 20 bytes long
    • has office_tel column that is 13 bytes long
    • has dept_no as primary key and system increments the key value by 1 automatically
    • does not allow NULL in any column except office_tel
    • does not allow duplicate values for dept_name column

Ex 5: Advanced query

Use ORDER BY, GROUP BY, HAVING, LIMIT, LIKE to :

  • search for all students who has student_id that contains 2007 (LIKE)
  • search for all students and order them by student_id (ORDER BY)
  • group students by the major and search for the major where its students average year is greater than 3 (GROUP BY, HAVING)
  • search for only two students from Business Administration who has student_id that contains 2007 (LIMIT)
